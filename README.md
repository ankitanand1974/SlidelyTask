# Slidely Task 2 - Form Submission Application

This project consists of a frontend application built with VB.NET and a backend API built with Node.js and Express. It allows users to submit form data, view submissions, and delete submissions.

## Frontend (VB.NET)

### Features

1. **Create Submission Form**
   - Input fields for Name, Email, Phone Number, and GitHub Link
   - Stopwatch functionality to track time spent on the form
   - Submit button to send data to the backend

2. **View Submissions Form**
   - Display submitted data including Name, Email, Phone Number, GitHub Link, and Stopwatch Time
   - Navigation buttons to move between submissions (Previous and Next)
   - Delete button to remove the current submission

3. **Main Form**
   - Buttons to open Create Submission and View Submissions forms
